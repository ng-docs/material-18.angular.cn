可以用来过滤 MatButtonToggleHarness 实例列表的标准。

属性
名称 描述

checked: boolean

只查找已勾选的实例。

disabled: boolean

仅查找与给定禁用状态匹配的实例。

name: string | RegExp

只查找名字与指定值匹配的实例。

text: string | RegExp

只查找其文本内容匹配指定值的实例。
