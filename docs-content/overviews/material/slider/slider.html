像 <input type="range"> 一样，<mat-slider> 允许通过鼠标、触摸或键盘来从范围中选择一个值。

选择一个值

默认情况下，滑杆（slider）的最小值是 0，最大值是 100，滑块（thumb）移动时的增量是 1。 这些值可以通过设置 min、max 和 step 属性进行修改。除非另行指定，否则其初始值就等于最小值。

<mat-slider min="1" max="5" step="0.5" value="1.5">
  <input matSliderThumb>
</mat-slider>

选择范围

A <mat-slider> can be converted into a range slider by projecting both a matSliderStartThumb and a matSliderEndThumb into it. Each thumb has its own value, but both are still constrained by the slider's min and max values. The matSliderStartThumb cannot have a value greater than that of the matSliderEndThumb and the matSliderEndThumb cannot have a value less than that of the matSliderStartThumb, though they both may have the same value.

<mat-slider>
  <input matSliderStartThumb>
  <input matSliderEndThumb>
</mat-slider>

滑块标签

默认情况下，滑块的精确选择值对用户不可见。但是，可以通过添加 discrete 属性将此值添加到滑块的滑块上。

<mat-slider discrete>
  <input matSliderThumb>
</mat-slider>

格式化滑块标签

默认情况下，滑块标签上的值会始终和模型的值相同，不过它可能太大，而没办法放进标签中。 如果你要控制要显示的值，你可以使用输入属性 displayWith 来实现。

刻度标记

默认情况下，滑块不会在滑块轨道上显示刻度线。可以使用 showTickMarks 属性启用此功能。

<mat-slider showTickMarks>
  <input matSliderThumb>
</mat-slider>

键盘交互

滑杆具有下列键盘绑定：

Key Action
Right arrow Increment the slider value by one step (decrements in RTL).
Up arrow Increment the slider value by one step.
Left arrow Decrement the slider value by one step (increments in RTL).
Down arrow Decrement the slider value by one step.
Page up Increment the slider value by 10% (of the max value).
Page down Decrement the slider value by 10% (of the max value).
End Set the value to the maximum possible.
Home Set the value to the minimum possible.

无障碍性

MatSlider 实现了 ARIA role="slider" 模式，它会处理键盘输入和焦点管理。始终要通过 aria-label 或 aria-labelledby 为每个滑块提供无障碍标签。
